# Commercial Shotlist Authoring Guide

Use the smallest structure that fits the edit:

```text
A + B + C
```

Use optional modifiers only when the edit needs proof inserts:

```text
A + B + M + C
```

For longer edits, include more than one modifier group:

```text
A + B + M1 + M2 + C
```

Modifiers are for client-requested details, footage-specific moments, longer edits, technical proof, process proof, and warranty or guarantee inserts.

Modifiers are not normal benefits. Benefits sell the reason. Modifiers prove the process.

Each modifier group needs a `label` and at least one variation with `text`. `usage`, variation `label`, and variation `notes` are optional.
