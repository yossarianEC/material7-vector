Instrucciones de grabación

Hable despacio y con pausas. Deje un segundo de silencio antes y después de cada frase para facilitar los cortes en edición.

Notas visuales

  • Mostrar el software o preparación de corte Immortal Cut cuando se use el modificador M1.
  • Mostrar instalación cuidadosa sobre el vehículo cuando se use el modificador M2.
  • Usar close-ups de pintura, bordes, puntos de contacto y acabado transparente.
  • Cerrar con beauty shot del auto terminado, logo, teléfono y ubicación.

Notas de audio

  • Grabar A, B, M y C como clips independientes.
  • Usar modificadores solo cuando el edit necesite más duración o cuando el footage muestre ese proceso.
  • Mantener tono natural, claro y profesional.
