Fundamentals

What a script is

A script is a decision document. It predicts every intentional second of screen time and resolves indecision before a camera is touched. It is not the video. It is the contract the video must honor.

4 laws

Law 1

One message

One takeaway. If there are two messages, there are zero messages. Write the single sentence this video must leave behind before writing anything else.

Law 2

The hook earns the rest

The first 3 seconds decide whether the remaining seconds exist. Assume the viewer will leave. Write the hook last, after you know what you're earning attention for.

Law 3

Every scene justifies itself

Remove anything that doesn't advance the message or the emotion. If it's nice to have, it doesn't belong in the script.

Law 4

Structure before style

The scaffold must be sound before any creative layer touches it. Voice, tone, and aesthetics solve nothing if the structure is wrong.

Universal anatomy

HookThe reason to keep watching. Not an intro — a disruption. Creates immediate tension or curiosity.
PayloadThe single message, delivered. The thing we need them to understand or feel. One beat only.
ResolutionClosure. The brand's answer or the emotional landing. Every open loop closes here.
CTAWhat happens next. Always specific. Implicit in emotional formats, but never absent in direct response.

Format specs

Choose the container before writing

Duration is not decoration. It determines how much context, tension, and closure the script can responsibly carry.

Short-form

Short-form

:15

~37 words spoken

Hook → Payload → Implicit CTA

No setup. No backstory. Hook and message are the same beat. CTA is visual or implied because there is no room to speak it cleanly.

Short-form

:30

~75 words spoken

Hook → Context → Payload → CTA

Minimal context only. One beat of setup if essential. CTA must be spoken and specific.

Short-form

:60

~150 words spoken

Hook → Setup → Tension → Payoff → CTA

First format where a real problem can be established. Tension must be earned, not decorative. Full CTA required.

Long-form

Long-form

:90

~225 words spoken

Hook → Context → Tension → Resolution → CTA

First format where a full arc fits. Resolution must feel earned, not handed to the viewer.

Long-form

2:00

~300 words spoken

Hook → Establish → Build tension → Climax → Resolution → CTA

Full story arc. A climax must exist — a point of maximum stakes. Not justified for product demos alone.

Long-form

3:00+

~450+ words spoken

Full narrative or brand film arc

Premium tier only. Every additional second must justify its cost. Brand film, manifesto, or deep demo only.

Word counts are based on a comfortable 150 WPM voiceover pace. Adjust ±15% for pacing intent. Always time the script aloud.

Script types

Pick one structural engine

Hybrid scripts are allowed only when the structural logic is obvious. If the format needs explanation, it is probably hiding confusion.

Problem → Solution

Pain established → Brand enters → Outcome shown

Most direct. Works at any duration. Problem must be real and felt, not manufactured. Risk: generic if the pain isn't specific enough.

Before → After

Contrast state A → Bridge → State B

Visual-first structure. The contrast does the work. Brand is the mechanism of transformation, not the subject. Best at :30–:60.

Product demo

Context → Feature shown → Result demonstrated

Show it doing the thing. Never describe what the viewer can already see. Each feature must solve something. Works best at :60+.

Testimonial structure

Credibility signal → Specific claim → Outcome → Brand

The specific claim is everything. “I loved it” is not a claim. “I closed three deals the same week” is. Script the structure, not the words.

Narrative / Story arc

Character → Conflict → Resolution

Brand is part of the resolution, not the subject of the story. Requires :90+. Character must be recognizable, not aspirational to the point of alien.

Talking head / Direct address

Authority → Problem → Offer → CTA

Speaker is the asset. Authority must be earned in the first 5 seconds or the format collapses. Do not cast anyone who needs warmup time.

Montage + VO

Visual rhythm carries emotion → VO carries meaning

VO and visuals must be complementary, not redundant. If VO describes what's on screen, one of them is wrong.

Pre-write

Resolve before writing

This checklist prevents the most expensive mistake in production: discovering the real strategy after the script is already written.

Message

Single message defined

One sentence. If it takes more than one, it isn't defined yet.

Hook strategy decided

Question, contrast, statement, shock, or visual. Pick one.

CTA is specific

Not “learn more.” What exactly happens after viewing?

Format

Duration confirmed

From the approved format list. Not approximate.

Platform confirmed

Platform determines aspect ratio, pacing, and hook tolerance.

Script type selected

From the 7 types. Hybrid only with clear structural logic.

Audience + context

Viewer defined

One specific person, not a demographic. Who is sitting there, watching this?

Viewing context defined

Scrolling at work? Sitting at home? Context shapes hook and pacing.

Assets + production

Available assets inventoried

Existing footage, talent, locations, brand elements — confirmed before scripting visual scenes.

Brand voice reference confirmed

An actual example of how this brand sounds. Not guidelines — real content.

Tone defined

One word or a short phrase. Constrains every word choice from here forward.

Quality check

Run before production

This is the final structural gate. If the script fails here, production does not fix it. Production only makes the mistake more expensive.

Structure

Hook lands in ≤3 seconds

Read the first line. Does it create tension or curiosity immediately? If not, rewrite.

Single message confirmed

Read the script. Count the messages. If there's more than one, decide which survives.

Every scene has one job

For each scene, write one word describing its job. If you can't, cut it.

Ending is definitive

The script closes. No ambiguity unless ambiguity is deliberate and written as strategy.

Copy

Opens on viewer, not brand

First word is not “we,” “our,” or the brand name. The viewer must see themselves first.

Zero filler lines

Read every line. Ask: what does this advance? If the answer is nothing, cut it.

CTA is specific and actionable

Exact phrase. Exact action. No interpretation required from the viewer.

Timing + alignment

Reads aloud at target duration

Time it out loud. Not in your head. Target tolerance is ±5%.

Visual and audio are complementary

VO should not describe what the viewer can already see. Each channel carries different information.

Transitions are logical

Each scene leads to the next. No scene appears without a reason for its position in the sequence.
